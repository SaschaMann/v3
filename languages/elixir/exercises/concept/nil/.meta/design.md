# Design

## Goal

The goal of this exercise is to teach how to use nil values for when nothing exists as a value.

## Learning objectives

- Know what `nil` represents
- Know that `nil` is a special atom
- Know how to pattern match on `nil`

## Out of scope

## Concepts

- `nil`
- `if-conditional`

## Prerequisites

- `strings`
